Food Delivery Analytics Using SQL is a database analytics project designed to analyze customer behavior, restaurant performance, order trends, delivery operations, and payment insights within a food delivery platform. The project uses a relational database consisting of multiple interconnected tables, enabling comprehensive SQL-based analysis and business reporting.

The dataset contains information about 500 customers, 100 restaurants, 500 orders, 500 menu items, 1,492 order items, 100 delivery partners, delivery records, and payment transactions. By leveraging SQL concepts such as Joins, Aggregate Functions, Subqueries, CTEs, Window Functions, and Views, the project extracts meaningful insights from operational data.

Key Areas of Analysis
* Customer acquisition and ordering behavior
* Restaurant performance and ratings analysis
* Revenue and sales trend analysis
* Popular menu items and cuisine preferences
* Order status and cancellation tracking
* Delivery partner performance evaluation
* Delivery time and operational efficiency analysis
* Payment method and transaction analysis

Technologies Used
* MySQL
* SQL
* GitHub
* CSV Dataset


This project demonstrates practical SQL skills by transforming raw food delivery data into actionable business insights. It showcases proficiency in database design, data querying, performance analysis, and business intelligence reporting, making it an excellent portfolio project for aspiring Data Analysts and SQL Developers.
